# Donate

FreedomOS includes a built-in donation workflow, but this public repository intentionally does not hardcode bank details or platform account links.

After installation, a superadmin can configure donations at:

```text
/admin/donations
```

Supported donation options:

- manual transfer instructions
- account reference notes
- popular donation platform links
- custom donation headline and copy

For the upstream project, use the repository contact channels to ask about current donation options:

- https://github.com/biolashonde/freedom_os
- https://github.com/biolashonde/freedom_os/issues

Please do not add private banking details or personal payment credentials directly to the repository.
